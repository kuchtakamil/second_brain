# Spring Boot od 2014

**Data:** 2024-11-22

---

**User:**

Od którego mniej więcej roku można używać określenia Spring Boot zamiast Spring.

---

**Assistant:**

Spring Boot oficjalnie zadebiutował w **2014 roku**, kiedy to wydano wersję **1.0**. Od tego momentu Spring Boot stał się popularnym podejściem do tworzenia aplikacji w ekosystemie Spring, upraszczając konfigurację i uruchamianie aplikacji.

Jeśli więc mowa o Spring Boot, w kontekście Spring można go używać od 2014 roku. Jednak "Spring" jako framework istnieje od 2003 roku i wciąż jest podstawą działania Spring Boot. Warto rozróżnić, że:

- **Spring** to ogólny framework zapewniający bogaty zestaw funkcji (np. Spring Framework).
- **Spring Boot** to narzędzie nadbudowane na Spring Framework, które ułatwia tworzenie aplikacji, eliminując dużą część konfiguracji i automatyzując wiele procesów. 

Od 2014 roku, w większości nowych projektów mówi się głównie o Spring Boot, bo dominuje on w nowoczesnych aplikacjach opartych na Spring.
