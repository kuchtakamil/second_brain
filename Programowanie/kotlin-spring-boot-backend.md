# Kotlin Spring Boot Backend

**Data:** 2024-11-10

---

**User:**

Chcę programować aplikacje back-endowe w języku Kotlin. Nie chcę pisać w Kotlinie aplikacji mobilnych. Czy pisząc aplikacje back-endowe, powszechną praktyką jest używanie frameworka Spring Spring Boot razem z Kotlin?

---

**Assistant:**

User wants to program back-end applications in Kotlin and is not interested in writing mobile applications in Kotlin.

---

**Assistant:**

Tak, używanie Kotlin wraz z frameworkiem Spring Boot jest popularne i w pełni wspierane w środowisku back-endowym. Kotlin dobrze współpracuje ze Spring Boot, co pozwala na zwięzły, bezpieczny typowo kod, a Spring ma pełną obsługę Kotlin. Kotlin oferuje nowoczesne funkcje, takie jak coroutines, które mogą ułatwiać asynchroniczne przetwarzanie danych.

Wiele firm i programistów wybiera Kotlin zamiast Javy, by zyskiwać na czytelności kodu, jego zwięzłości oraz funkcjach typowych dla nowoczesnych języków.
