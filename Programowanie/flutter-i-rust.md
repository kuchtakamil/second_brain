# Flutter i Rust

**Data:** 2025-01-17

---

**User:**

Jaki język programowania wykorzystuje Flutter? Może Rust?

---

**Assistant:**

Flutter wykorzystuje **Dart** jako główny język programowania. Rust nie jest natywnie obsługiwany przez Fluttera, ale można go używać w formie natywnych bibliotek skompilowanych do kodu maszynowego i wywoływanych przez FFI (Foreign Function Interface).  

Jeśli interesuje Cię integracja Fluttera z Rustem, możesz użyć np. **flutter_rust_bridge**, który ułatwia komunikację między kodem Dart a kodem napisanym w Rust.
