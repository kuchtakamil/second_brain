# Znajdź pull requesty z tekstem

**Data:** 2023-10-02

---

**User:**

GitHub, chcę znaleźć pull requesty w których modyfikacje zawierają określony tekst

---

**Assistant:**

Aby znaleźć pull requesty na GitHubie, w których modyfikacje zawierają określony tekst, można skorzystać z funkcji wyszukiwania na platformie GitHub. Oto kroki, które można podjąć:

1. Przejdź na stronę internetową GitHub: https://github.com/.

2. W prawym górnym rogu znajdź pole "Wyszukaj lub przejdź do..." i wpisz swoje kryteria wyszukiwania. W tym przypadku możesz użyć takiej frazy:

```
is:pr is:open "twój tekst"
```

Gdzie:
- `is:pr` oznacza, że szukasz pull requestów.
- `is:open` oznacza, że szukasz tylko otwartych pull requestów.
- `"twój tekst"` to tekst, który chcesz znaleźć w modyfikacjach.

3. Naciśnij klawisz Enter lub kliknij ikonę lupki, aby rozpocząć wyszukiwanie.

Wyniki wyszukiwania pokażą listę pull requestów, które zawierają określony tekst w swoich modyfikacjach. Możesz kliknąć na interesujący cię pull request, aby przejrzeć szczegóły i zobaczyć, gdzie dokładnie znajduje się ten tekst w zmianach.

Pamiętaj, że GitHub może być aktualizowany, a procedury wyszukiwania mogą ulec zmianie. Dlatego warto sprawdzić oficjalną dokumentację GitHuba lub przegląd instrukcji dostępnych na platformie, jeśli potrzebujesz bardziej szczegółowych informacji na temat wyszukiwania pull requestów.
