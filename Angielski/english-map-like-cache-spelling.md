# English: "Map-like Cache Spelling"

**Data:** 2023-10-02

---

**User:**

Czy pisząc po angielsku 'a map like cache' powinienem napisać like razem czy osobno? Chodzi o to że cache przypomina mapę


---

**Assistant:**

W języku angielskim, zwrot "a map like cache" powinien być zapisywany razem, czyli "a map-like cache". To oznacza, że cache przypomina lub jest podobny do mapy.
