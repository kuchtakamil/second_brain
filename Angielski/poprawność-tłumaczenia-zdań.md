# Poprawność tłumaczenia zdań

**Data:** 2025-12-15

---

**User:**

Zapisz cały kod z filmu.
Write down all the code from the movie - czy to poprawnie?

---

**Assistant:**

Nie do końca.

**„Write down all the code from the movie”** jest zrozumiałe, ale brzmi nienaturalnie po angielsku, bo **write down** sugeruje ręczne notowanie, a **from the movie** jest nieprecyzyjne.

Lepsze opcje, zależnie od sensu:

- **„Write out all the code shown in the movie.”** – najbardziej naturalne  
- **„Transcribe all the code from the movie.”** – jeśli chodzi o dokładne przepisanie  
- **„Copy all the code shown in the video.”** – jeśli to film instruktażowy / tutorial  
- **„Write down all the code shown in the video.”** – poprawne, ale mniej naturalne niż *write out*

Jeśli chcesz, podaj kontekst (film fabularny vs. tutorial), wtedy można dobrać najtrafniejsze zdanie.
