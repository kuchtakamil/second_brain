The sale of perfumes by private individuals takes place in Facebook groups. Sellers post in these groups announcing that they have perfumes for sale: new perfumes, partially used bottles, decants (perfumes poured into smaller bottles, e.g., 5ml), as well as manufacturer’s samples. Buyers browse the posts on Facebook and contact the sellers via Messenger or in the comments under the posts.  

**Problem the application aims to solve:** Browsing Facebook posts and searching for the perfumes one wants to buy is troublesome. It is difficult to find the desired perfumes. Old posts are quickly replaced by new ones, so in order to see what is available for sale, one must scroll through many pages/posts. It is not possible to distinguish whether one is looking for a new bottle, a partially used one, a decant, or a sample.  

In the future, the application is also intended to address the issue of counterfeit sales. In the first stage, verification of perfumes put up for sale will be possible by a human moderator. Later, the application could use an LLM model to determine, based on photos, whether the product is counterfeit.  

**Application features:**  
1. A user can register or log in.  
2. Each user can be both a Buyer and a Seller. No special permissions are required to become a Seller in the application.  
3. As a Buyer, a user can browse perfumes listed for sale. They can also search for perfumes by name, with filters for new perfumes, partially used bottles, decants, or samples.  
4. As a Seller, a user can list perfumes for sale as new, partially used, decants, or samples. Each listing must include the price, volume, photo, and optionally a description.  
5. As a Buyer, a user can request verification of a perfume’s authenticity in a listing by a moderator based on a photo. The moderator’s assessment is visible in the listing to all users.  
6. There is a seller rating system.  
7. Each perfume (or rather each listing) has a reference to its profile on fragrantica.com. Users can click the link to fragrantica to learn more about the perfume’s profile.  
8. Sellers can add listings using a dedicated ad creation panel.  
9. The application’s author earns revenue from contextual ads. Ads must not be intrusive or disruptive to the user experience.
